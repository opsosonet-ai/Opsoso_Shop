Application Restart Guide
Windows Restart Instructions
Method 1: IDE Restart (Recommended)
  1. If running in IntelliJ IDEA or Eclipse:
    • Stop the running application (Red square button)
    • Click "Run" or "Debug" again
Method 2: Command Line
  1. Open Command Prompt or PowerShell as Administrator
  2. Navigate to your project directory
  3. Stop the application: Ctrl + C
  4. Restart with: mvn spring-boot:run
Method 3: Task Manager
  1. Open Task Manager (Ctrl + Shift + Esc)
  2. Find Java processes and end them
  3. Restart the application
Linux Restart Instructions
Method 1: Terminal (Current Session)
# Stop current application
Ctrl + C

# Restart application
mvn spring-boot:run
Method 2: Kill Process
# Find Spring Boot process
ps aux | grep spring-boot

# Kill the process
pkill -f spring-boot

# Restart application
mvn spring-boot:run
Method 3: Service Mode (if configured)
# Restart as service
sudo systemctl restart your-app

# Check status
sudo systemctl status your-app
Important Notes
  • Configuration Update: The application.properties file has been updated with new database settings
  • Restart Required: Changes will only take effect after a full application restart
  • Port 8080: Make sure port 8080 is available before restarting
  • Database Connection: Ensure your database server is running and accessible
After Restart - Test Connection

Once the application restarts successfully, you can:

  1. Access the homepage: http://localhost:8080
  2. Check settings page: http://localhost:8080/settings
  3. Verify database connection in the application logs

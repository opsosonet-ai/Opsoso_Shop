Windows Instructions
Common Windows MariaDB/MySQL Settings:
  • XAMPP: Username: root, Password: (usually empty)
  • WAMP: Username: root, Password: (usually empty)
  • Laragon: Username: root, Password: (usually empty)
  • Manual Install: Check your installation notes
Tip: If you forgot your password, you can reset it using MySQL/MariaDB command line tools.
Linux Instructions
Common Linux MariaDB/MySQL Commands:
# Check MariaDB status
sudo systemctl status mariadb

# Connect to MariaDB
mysql -u root -p

# Create user for application
CREATE USER 'oss_app'@'localhost' IDENTIFIED BY 'your_password';
GRANT ALL PRIVILEGES ON oss.* TO 'oss_app'@'localhost';
FLUSH PRIVILEGES;
